# Pygros

An open-source fanmade chart emulator of rhythm game [Phigros](https://pigeon-games.com/phigros). 

## Requirements

[Cocos2d(Python)](http://python.cocos2d.org/)

Install it by running:
```shell script
pip install cocos2d
```

## Demo

```shell script
python demo.py
```
